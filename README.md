# agentix-website

Static multi-page website rebuild for Agentix Technology Advisors.

## Pages

- `index.html` - home page
- `services.html` - services overview
- `ai-transformation.html` - AI transformation page
- `about.html` - firm overview
- `contact.html` - contact page

## Assets

- `assets/css/styles.css` - shared site styling
- `assets/js/main.js` - mobile navigation and reveal interactions
- `assets/images/` - image assets
- `robots.txt` - crawler rules
- `sitemap.xml` - page sitemap

## Local preview

Serve the folder with any static file server, for example:

`python -m http.server 4173`
